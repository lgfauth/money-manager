## ?? Comandos Rápidos

```bash
# Executar todos os testes
dotnet test

# Com verbosidade
dotnet test --logger "console;verbosity=detailed"

# Com cobertura
dotnet test --collect:"XPlat Code Coverage"

# Watch mode
dotnet watch test

# Testes específicos
dotnet test --filter "FullyQualifiedName~TransactionServiceTests"
```
